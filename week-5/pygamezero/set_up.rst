Pygame Zero setup
-------------

**Step 2:** follow the Pygame Zero tutorials

https://pygame-zero.readthedocs.io/en/stable/introduction.html

**Step 3:** create something!

You can download free game assets from https://kenney.nl/assets?q=2d
